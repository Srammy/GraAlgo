# 散列函数
无论给散列函数什么数据，它都返回一个数字；将输入映射到数字。

需满足的要求：

1.将同样的输入映射到相同的索引。

2.将不相同的输入映射到不同的索引。

3.知道数组有多大，只返回有效的索引。

#  散列表
散列函数和数组得到散列表

速度很快

适用于：

1.仿真映射关系。

2.防止重复

3.缓存数据

# 性能
冲突：两个键分配相同的位置

发生冲突时，处理方式有很多，最简单的就是：在一个位置存储一个链表。

为避免冲突，需要有：

1.较低的填装因子

填装因子越低，发生冲突的可能性越小，散列表性能越好。一个经验：填装因子大于0.7，就调整散列表的长度

2.良好的散列函数

良好的散列函数让数组中的值呈均匀分布
